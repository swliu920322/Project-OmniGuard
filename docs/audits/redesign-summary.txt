🎯 核心改动内容

✅ 已完成:

  1. 安全性增强
     • OpenAI API Key 不再出现在前端
     • 凭证完全存储在 local.settings.json（本地文件）
     • 凭证通过环境变量注入到后端

  2. 代码简化
     • LaunchConfig 接口移除不必要的字段
     • 后端不再接受前端的参数覆盖
     • 前端请求体从 4 个字段减少到 2 个

  3. 脚本整理
     • 将复杂的 infra-up.sh 拆分为主题清晰的脚本
     • 从 7 个脚本精简到 5 个核心脚本
     • 每个脚本职责单一，易于维护

  4. Makefile 重写
     • 从 1 个 Makefile 有 8 个目标（还有重复）到 6 个清晰目标
     • 添加 help 目标，用户可快速查看所有命令
     • 所有命令都有明确的设置权限

  5. 文档完善
     • QUICKSTART.md: 新用户快速上手
     • ARCHITECTURE_REDESIGN.md: 详细改动说明
     • COMMAND_REFERENCE.md: 命令速查表
     • 各脚本都有详细的中文注释

🚀 用户现在可以:

  # 工作流 1: 本地开发（最常用）
  $ make provision        # 一次性：部署 Azure 基础设施
  $ make start            # 日常：启动前后端
  $ make destroy          # 完成：清理资源

  # 工作流 2: 本地开发 + 上云
  $ make provision
  $ make start            # 开发...
  $ make deploy-function  # 代码完成，上云
  $ make trigger-ci       # 触发自动化流程
  $ make destroy
